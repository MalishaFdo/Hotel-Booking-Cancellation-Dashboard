# Hotel-Booking-Cancellation-Dashboard
The Hotel Booking Cancellation Dashboard provides a powerful, visual representation of patterns, risks, and customer behaviours associated with booking cancellations. This data product enhances decision-making for hotel managers and analysts by delivering insights into cancellation trends, risk factors, customer segmentation, and price sensitivity using interactive visual tools. This Power BI dashboard was developed to tell a clear and actionable story: why do hotel cancellations occur, and how can we proactively manage them? With growing pressure on hotels to predict and react to booking behaviours, this dashboard leverages machine learning and segmentation to support smarter decisions.
